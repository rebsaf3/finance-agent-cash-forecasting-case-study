# 05 Forecast Method And Assumptions
This case study assumes a simple cash waterfall by date.

Per date
Beginning balance
Plus inflows
Minus outflows
Equals ending balance

Assumptions are owned by finance.
The agent must not change them without explicit instruction.

Rules
1. Estimates must be labeled
2. Actuals must be labeled
3. Confidence notes must explain what is estimated and why
4. If assumptions are missing, the agent must produce a gap report
