# 03 Goals And NonGoals
Goals
1. Produce a consistent cash forecast output
2. Make inputs explicit and traceable
3. Validate before calculating
4. Refuse ambiguous scope through selection and disambiguation
5. Explain variance drivers with source grounded evidence
6. Support audit review through run logs

Non goals
1. Autonomous payments or banking actions
2. Replacing finance ownership of assumptions
3. Guessing missing values to keep the forecast moving
4. Hiding uncertainty behind confident language
