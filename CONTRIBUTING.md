# Contributing
This repo is a public case study. Contributions are welcome if they improve clarity, safety, and usefulness without adding sensitive details.

Rules
1. Do not add tenant names, tenant ids, internal urls, secrets, tokens, or real financials
2. Do not add proprietary policies, bank details, or account identifiers
3. Keep examples sanitized and illustrative only
4. If you add a new claim, add or update a test that validates the behavior

Process
1. Open an issue using the templates
2. Propose the change with the impacted files
3. Submit a pull request referencing the issue
4. Update governance/Change_Log.md when behavior changes
